# Tokamak Particle Simulation

Simulation and control of a charged particle in a toroidal magnetic field inspired by tokamak fusion reactors.

## 📌 Context
This project was developed as part of the course **"Modelación de Procesos"** in the **Civil Electronic Engineering program at Universidad de Concepción (UdeC), Chile**.

The objective was to model, analyze, and control a physical system using mathematical tools and simulation environments.
## 📌 Description
This project models the nonlinear dynamics of a charged particle under a magnetic confinement system. The system is analyzed and controlled using classical control techniques.

## ⚙️ Features
- Nonlinear dynamic model (Lagrangian formulation)
- Linearization around equilibrium point
- Transfer function derivation
- Closed-loop control design
- Simulation in MATLAB/Simulink

## 🧠 Concepts
- Dynamical systems
- Control systems
- Magnetic confinement
- Numerical simulation

## 🛠️ Tools
- MATLAB
- Simulink
